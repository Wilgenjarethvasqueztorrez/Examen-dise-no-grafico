Principios de Diseño Gráfico

A continuación se presentan algunos de los principios fundamentales del diseño gráfico:

  • Proximidad: Agrupa elementos relacionados para crear una conexión visual.
  • Balance: Equilibra los elementos en la página para que no parezca pesada de un lado.
  • Alineación: Mantén una alineación consistente para crear una sensación de orden.
  • Contraste: Usa contrastes para resaltar elementos importantes.
  • Repetición: Repite elementos de diseño para crear cohesión.
  • Espacio en blanco: Usa espacio en blanco para dar a los elementos "espacio para respirar".
